# Frontend Governance

This directory is for repository-wide frontend design and frontend implementation guardrails.

## Use This Directory For

- frontend visual-governance rules
- page-pattern and interaction-governance guidance
- TDesign workflow or frontend AI-assist governance

## Boundaries

- Keep reusable page-template references in `../../graft-design-system/`.
- Use `../../domains/` when the document mainly describes one business capability instead of shared frontend rules.
